# order-made-designs

オーダーメイド工房の **製造データ・制作記録** リポジトリ。

レーザー加工・革製品の設計データと、制作・販売の方針をまとめています。

---

## 販売方針

| 段階 | 売り場 | 内容 |
|------|--------|------|
| **量産前（試作〜小ロット）** | **メルカリ** | 試作品の販売・反応確認 |
| **量産後** | **Amazon** | メイン販売 |
| **名入れ・オーダー** | **自社サイト** | 名入れの受付・相談 |

- **名入れ** → 自社サイト（[問い合わせ](http://54.65.228.227/contact.html)）で受け付け
- **試作期の購入** → メルカリ
- **量産開始後** → Amazon を本番の売り場とする

---

## 制作の流れ

### 試作フェーズ（手作りで完成まで）

```
① 本体3Dでサイズ確認
      ↓
② どういうものを作るか考える（デザイン）
      ↓
③ 革の型を作成
      ↓
④ 手で革を切って試作品を作る
      ↓
   ②③④ を繰り返す → 完成
```

### 量産フェーズ（完成後）

```
⑤ レーザーカット用の2D図面を作成
      ↓
⑥ 複数枚の革を固められる型（量産用ジグ）を作る
```

**レーザー2D・量産用ジグは、試作が完成してから作る。**
