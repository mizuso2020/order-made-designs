---

## 各ファイルの役割

### ① 本体 3D（`3d-device/`）
- AirTag **実物の形** のデータ
- ケース設計時の **サイズ取り・はめ込み確認** 用
- **商品ではない**（参考データ）

### ② ケース 3D（`3d-case/`）
- **売るケース** の3Dデータ
- 3Dプリントまたは設計の元データ

### ③ レーザー 2D（`laser/`）
- レーザー加工機用の **2D図面**
- 穴あけ・彫刻・一部裁断

### ④ 革の型（`leather/`）
- **革を裁断するための型紙**
- ケースの外皮パターン
- ケース3D（`3d-case/`）とは別データ
