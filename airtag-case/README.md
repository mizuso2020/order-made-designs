商品ごとに、必要なファイルをフォルダ分けして管理します。

## フォルダ構成（ルール）

```
order-made-designs/
├── README.md
└── airtag-case/
    ├── README.md
    ├── 3d-device/       ← ① 本体3D（サイズ確認）
    ├── leather/         ← ③ 革の型（試作中に更新）
    ├── laser/           ← ⑤ 量産用レーザー2D（完成後）
    └── jig/             ← ⑥ 量産用ジグ（完成後）
```

### ファイル名のルール

```
airtag-device-v1.stl    … 本体（参考）
airtag-case-v1.stl      … ケース
airtag-case-v1.svg      … レーザー2D
airtag-case-v1.dxf      … 革の型
```

- **英小文字 + ハイフン** のみ（スペース・日本語不可）
- 改良したら **v2, v3…**（古い版は残す）

---

## 新しい商品を追加するとき

1. 商品フォルダを作る
2. `3d-device/` `3d-case/` `laser/` `leather/` と `README.md` を作る
3. この README の **商品一覧表** を更新
4. GitHub にアップロード

---

## 載せないもの

- 顧客の個人情報・名入れデータ
- 有料素材のパターン
- AWS 鍵ファイル（.pem 等）
