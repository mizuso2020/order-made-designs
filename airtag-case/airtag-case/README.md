## 販売方針

| 段階 | 売り場 | 内容 |
|------|--------|------|
| **量産前（試作〜小ロット）** | メルカリ | 試作品の販売・反応確認 |
| **量産後** | Amazon | メイン販売 |
| **名入れ・オーダー** | **自社サイト** | 名入れ内容の受付・相談 |

- 名入れは **サイト（問い合わせ）** で受け付ける
- 試作期の購入は **メルカリ**
- 量産開始後は **Amazon** を本番の売り場とする

## 制作の流れ（全体）

### 試作フェーズ

```
① 本体3Dでサイズ確認
      ↓
② どういうものを作るか考える（デザイン）
      ↓
③ 革の型を作成
      ↓
④ 手で革を切って試作品を作る
      ↓
   ②③④ を繰り返す → 完成
```

### 量産フェーズ（完成後）

```
⑤ レーザーカット用の2D図面を作成
      ↓
⑥ 複数枚の革を固められる型（量産用ジグ）を作る
```

詳細は各商品フォルダの README を参照（例：[airtag-case/README.md](airtag-case/README.md)）
