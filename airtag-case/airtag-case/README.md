# エアタグケース

Apple AirTag 用オーダーメイドケース（革 + レーザー）

---

## 制作の流れ（実際の工程）

### 試作フェーズ（手作りで完成まで）

```
① 本体3Dでサイズ確認
      ↓
② どういうものを作るか考える（デザイン）
      ↓
③ 革の型を作成
      ↓
④ 手で革を切って試作品を作る
      ↓
   ②③④ を繰り返す → 完成
```

### 量産フェーズ（完成後）

```
⑤ レーザーカット用の2D図面を作成
      ↓
⑥ 複数枚の革を固められる型（量産用ジグ）を作る
```

**レーザー2D・量産用ジグは、試作が完成してから作る。**

---

## データと工程の対応

| 工程 | 使うデータ | フォルダ |
|------|-----------|---------|
| ① サイズ確認 | AirTag本体3D | `3d-device/` |
| ③ 革の型 | 革パターン | `leather/` |
| ⑤ 量産用レーザー | 2D図面 | `laser/` |
| ⑥ 量産用ジグ | 複数枚固定用の型 | `jig/` |

※ `3d-case/` は3Dプリント版ケースを作る場合のみ使用

---

## 各ファイルの役割

### 本体3D（`3d-device/`）
- AirTag 実物の形（参考）
- サイズ取り・はめ込み確認用

### 革の型（`leather/`）
- 手裁ち試作用の型紙
- **試作フェーズで②③④を回すたびに更新**

### レーザー2D（`laser/`）
- **試作完成後**、量産用に書く図面
- レーザー裁断・彫刻用

### 量産用ジグ（`jig/`）
- 複数枚の革を重ねて固定する型
- 量産時のばらつき防止用
