# order-made-designs

オーダーメイド工房の **製造データ置き場** です。  
商品ごとに、必要なファイルをフォルダ分けして管理します。

## 関連リンク

- ショップサイト：http://54.65.228.227
- Webサイト（HTML）：https://github.com/mizuso2020/custom-shop-site

## 1商品に必要なファイル（4種類）

| 種類 | フォルダ | 形式 | 用途 |
|------|---------|------|------|
| **AirTag本体 3D** | `3d-device/` | .stl | サイズ取り・フィット確認用（参考） |
| **ケース 3D** | `3d-case/` | .stl | 3Dプリントするケース本体 |
| **レーザー 2D** | `laser/` | .svg .dxf | レーザー裁断・彫刻用 |
| **革の型** | `leather/` | .dxf .pdf | 革裁断用パターン |

※ 商品によって不要なフォルダは空のままでOK

## フォルダ構成（ルール）

```
order-made-designs/
├── README.md
└── airtag-case/
    ├── README.md
    ├── 3d-device/       ← AirTag本体（参考・サイズ取り）
    │   └── airtag-device-v1.stl
    ├── 3d-case/         ← ケース（3Dプリント・商品）
    │   └── airtag-case-v1.stl
    ├── laser/           ← レーザー裁断・彫刻（2D）
    │   └── airtag-case-v1.svg
    └── leather/         ← 革の型（裁断）
        └── airtag-case-v1.dxf
```

### ファイル名のルール

```
airtag-device-v1.stl    … 本体（参考）
airtag-case-v1.stl      … ケース
airtag-case-v1.svg      … レーザー2D
airtag-case-v1.dxf      … 革の型
```

- **英小文字 + ハイフン** のみ（スペース・日本語不可）
- 改良したら **v2, v3…**（古い版は残す）

---

## 新しい商品を追加するとき

1. 商品フォルダを作る
2. `3d-device/` `3d-case/` `laser/` `leather/` と `README.md` を作る
3. この README の **商品一覧表** を更新
4. GitHub にアップロード

---

## 載せないもの

- 顧客の個人情報・名入れデータ
- 有料素材のパターン
- AWS 鍵ファイル（.pem 等）
